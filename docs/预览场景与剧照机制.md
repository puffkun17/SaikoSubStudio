# 预览场景与剧照机制（SaikoSubStudio）

> 本文档记录 Theater 放映厅的完整预览场景（环境光效 + TV 边框 + 背景剧照）与“换张剧照”功能。基于 NAS SaikoBasement 完整代码 + 2026-06 重构补充。

## 核心概念

- **预览场景 (Preview Scene)**：由 `sceneBackground` 状态驱动（'cinema' | 'nature' | 'night' | 其他 solid）。
  - 影响 `BackdropSlot` 类型为 'preset' 时使用的背景图。
  - 结合画幅 (theaterAspect) 决定 TV 边框遮罩（4:3 用 crt 系列，其它用 modern 系列）。
- **剧照 (Still / Backdrop)**：
  - TMDB 自动/手动检索时获取 backdrops 或 episode stills，形成 `tmdbBackdropList` 池。
  - “换张剧照” = `shuffleBackdrop()` 从池中随机切换（排除当前）。
  - 也支持自定义（ColorSampler 上传的 refScreenshot 可扩展为 image 类型 backdrop）。
- **渲染**：`ScreenSimulator.getBackdropStyle()` 根据 `backdrop` prop 决定 backgroundImage 或 solid color。
  - preset 映射：
    - nature → /scene_nature.png
    - night → /scene_night.png
    - cinema / 默认 → /scene_portrait.png

## UI 控制点

- **ControlDeck**（放映厅顶部工具条）：
  - 画幅比例（4 种，自动切换 crt / modern 边框）。
  - 模拟场景（新增显式按钮：电影院 / 自然光 / 暗夜）。
  - 剧照标签 + “换张剧照”按钮（仅当 TMDB 池 >=2 张时出现）。
- **StyleSidebar**（样式参数抽屉）：
  - 高级隐藏设置：TMDB API Key 输入（密码框，blur 时调用 setTmdbApiKey 持久化到 localStorage）。
- **TmdbPanel + ColorSampler**：提供剧照来源（TMDB 检索池、上传参考截图）。
- **TheaterStep**：把 sceneBackground + tmdbBackdrop 转换为 `BackdropSlot` 传给 Simulator（优先 tmdb 覆盖 cinema）。

## 资产

- TV 边框（必须）：`tv-crt*.png`、`tv-modern*.png`（public/ 根）。
- 场景背景（重构时用 Grok image_gen 补全）：
  - `scene_portrait.png`（影院/默认，暗调戏剧性）
  - `scene_nature.png`（自然光，明亮）
  - `scene_night.png`（暗夜，氛围）
- 其他环境参考：web_images/ 下的 stadium 系列（可作为自定义剧照扩展）。

所有图片均在 `public/` 下，Next.js 自动 serve 为 `/scene_xxx.png`。

## 状态与持久化（store）

- `sceneBackground: 'cinema'`（默认）
- `tmdbBackdrop` + `tmdbBackdropList`
- `tmdbApiKey`：**已移除硬编码**（原代码两个 store 副本 + test 脚本均为硬编码 e9fa...，无法“提取”配置）。现在：
  - 初始 ''
  - `setTmdbApiKey(key)` → 同时写 localStorage('saiko_tmdb_api_key')
  - initializeLibrary 加载
  - 搜索函数前置 guard：无 key 时友好错误提示 + 引导去高级设置
- 切换场景或 shuffle 会同步更新当前 task（便于 library 保存/载入时恢复场景 + 剧照）。

## 历史与重构说明

- NAS SaikoBasement 完整版已有 preset 逻辑 + shuffle + stills/backdrops 区分（tv episode 用 stills）。
- 多次提取导致 UI 静态化（只有标签无切换按钮）、场景图片丢失 → “换张剧照修复后预览场景丢失”。
- 本次重构：
  - 恢复/新增场景显式选择器。
  - 用 Grok 图片生成技能补全 3 张场景图并转为 png 匹配原引用。
  - 彻底清理 tmdbApiKey 硬码，实现隐藏本地配置。
  - 保持纯客户端 + CF 兼容。
